# 🔐 Secure VPC Architecture on AWS

## 📌 Project Overview
This project demonstrates a secure AWS architecture using VPC with public and private subnets. The application is hosted on a private EC2 instance and exposed securely using an Application Load Balancer.

## ⚙️ Components Used

- VPC (Virtual Private Cloud)
- Public & Private Subnets
- Internet Gateway (IGW)
- NAT Gateway
- Bastion Host (Jump Server)
- EC2 Instance (Private)
- Nginx Web Server
- Application Load Balancer (ALB)
- Security Groups
